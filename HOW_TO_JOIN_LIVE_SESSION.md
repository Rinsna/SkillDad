# How Instructors Can Join Live Sessions

## Quick Guide for Universities/Instructors

### Step 1: Schedule a Session
1. Go to **University Dashboard** → **Live Sessions** tab
2. Click **"+ Schedule Session"**
3. Fill in details and click **"Schedule & Notify"**

### Step 2: Start the Session
1. Find your scheduled session
2. Click the **"Start"** button (changes status to LIVE)

### Step 3: Join as Instructor

You have **TWO ways** to join:

#### Option A: Enter Studio (Easiest)
1. Click the **"Enter Studio"** button
2. Opens session page with Zoom embedded
3. You join as host automatically

#### Option B: Host Link
1. Click the **"Host Link"** button
2. Copy the generated URL
3. Open URL in new browser tab
4. Join as host with full controls

### Step 4: End the Session
1. Click **"End"** button when done
2. Recording processes automatically

## Troubleshooting

**"Access Denied" Error:**
- We just fixed this! Refresh your browser
- Make sure you're logged in as the university that created the session

**"Enter Studio" Not Working:**
- Click "Start" button first to make session LIVE
- Then click "Enter Studio"
- If still not working, use "Host Link" option instead

**No Zoom Meeting:**
- Check that Zoom credentials are configured in backend
- Verify `ZOOM_MOCK_MODE=false` in server/.env
